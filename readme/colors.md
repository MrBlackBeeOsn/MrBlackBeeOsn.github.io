Màu nhạt hơn (Lightness ~80%):

Hex: #A8A4F2
RGB: (168, 164, 242)
HSL: (243, 76%, 80%)
Mô tả: Màu tím nhạt hơn, vẫn giữ sắc độ gốc nhưng sáng hơn.
